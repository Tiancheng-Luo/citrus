Class Citrus::ButPredicate

  1. lib/citrus.rb
Parent: Object

A ButPredicate is a Predicate that consumes all characters until its rule matches. It must match at least one character in order to succeed. The Citrus notation is any expression preceded by a tilde, e.g.: 

~expr
