Class Citrus::AndPredicate

  1. lib/citrus.rb
Parent: Object

An AndPredicate is a Predicate that contains a rule that must match. Upon success an empty match is returned and no input is consumed. The Citrus notation is any expression preceded by an ampersand, e.g.: 

&expr
