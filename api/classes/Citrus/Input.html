Class Citrus::Input

  1. lib/citrus.rb
Parent: Object

This class represents the core of the parsing algorithm. It wraps the input string and serves matches to all nonterminals.

Attributes

cache [R] A two-level hash of rule id’s and offsets to their respective matches. Only present if memoing is enabled.
cache_hits [R] The number of times the cache was hit. Only present if memoing is enabled.
max_offset [R] The maximum offset that has been achieved.
string [R] The input string.

match (rule, offset=0)

Returns the match for a given rule at offset. If memoing is enabled and a match does not already exist for the given rule/offset pair then the rule is executed and the result is cached before returning. See pdos.csail.mit.edu/~baford/packrat/icfp02/ for more information on memoing match results (also known as packrat parsing).
