Class Citrus::Label

  1. lib/citrus.rb
Parent: Object

A Label is a Predicate that applies a new name to any matches made by its rule. The Citrus notation is any sequence of word characters (i.e. [a-zA-Z0-9_]) followed by a colon, followed by any other expression, e.g.: 

label:expr
