Class Citrus::NotPredicate

  1. lib/citrus.rb
Parent: Object

A NotPredicate is a Predicate that contains a rule that must not match. Upon success an empty match is returned and no input is consumed. The Citrus notation is any expression preceded by an exclamation mark, e.g.: 

!expr
