Module Citrus::Proxy

  1. lib/citrus.rb

A Proxy is a Rule that is a placeholder for another rule. It stores the name of some other rule in the grammar internally and resolves it to the actual Rule object at runtime. This lazy evaluation permits us to create Proxy objects for rules that we may not know the definition of yet.
