Module Citrus

eval (code, options={})

Evaluates the given Citrus parsing expression grammar code in the global scope. Returns an array of any grammar modules that are created. 

Citrus.eval(<<CITRUS)
grammar MyGrammar
  rule abc
    "abc"
  end
end
